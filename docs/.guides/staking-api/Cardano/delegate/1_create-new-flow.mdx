To initiate the delegation process, create a new flow with a `POST` request to `/api/v1/flows`.

### Parameters

- `flow` &mdash; `object`
  - `network_code` &mdash; `string` Network on which this flow operates (ex. `cardano`) (required).
  - `chain_code` &mdash; `string` Chain on which this flow operates (ex. `preprod`) (required).
  - `operation` &mdash; `string` The operation to perform (ex. `staking`) (required).
  - `version` &mdash; `string` API version to use (ex. `v1`) (optional).

#### Sample cURL Request

```bash {5-10}
curl -X POST 'https://cardano-slate.datahub.figment.io/api/v1/flows' \
-H 'Content-Type: application/json' \
-H 'Authorization: <api_key>' \
-d '{
  "flow": {
    "network_code": "cardano",
    "chain_code": "preprod",
    "operation": "staking",
    "version": "v1"
  }
}'
```

### Response

- `id` &mdash; `string` &mdash; ID of the newly created flow.
- `state` &mdash; `string` &mdash; The current state of the flow.
- `actions` &mdash; `array` &mdash; An array of possible actions.

The response indicates 1 possible action:

- `assign_staking_data` &mdash; Assign the delegator address and validator (staking pool) address.

The `assign_staking_data` action indicates 2 fields for data collection:

- `delegator_address` &mdash; The wallet from which the ADA will originate.
- `validator_address` &mdash; The public key of the validator (staking pool) to which the ADA will be delegated.
