## Introduction

There are two ways of delegating SOL:

- Using an existing stake account that is in an inactive state
- Generating a new stake account and funding it with SOL

In the following guide we will illustrate how to create a new stake account.

:::info note

If you already have a stake account, you may reuse it.

:::
