---
title: Create a New Redelegation Flow
slug: /guides/staking-api/cosmos/redelegate/create-new-flow
---

To initiate the redelegation process, create a new flow with a `POST` request to `/api/v1/flows`.

### Parameters

- `flow` &mdash; `object`
  - `network_code` &mdash; `string` Network this flow operates on (ex. `cosmos`) (required).
  - `chain_code` &mdash; `string` Chain this flow operates on (ex. `testnet`) (required).
  - `operation` &mdash; `string` The operation to perform (ex. `redelegate`) (required).
  - `version` &mdash; `string` API version to use (ex. `v1`) (optional).

#### Sample cURL Request

```bash {5-10}
curl -X POST 'https://cosmos-slate.datahub.figment.io/api/v1/flows' \
-H 'Content-Type: application/json' \
-H 'Authorization: <api_key>' \
-d '{
  "flow": {
    "network_code": "cosmos",
    "chain_code": "testnet",
    "operation": "redelegate",
    "version": "v1"
  }
}'
```

### Response

- `id` &mdash; `string` ID of the newly created flow.
- `state` &mdash; `string` The current state of the flow.

The response indicates one possible action:

- `create_redelegate_tx` &mdash; Use this action to generate a redelegation transaction to stake your ATOM.

The `create_redelegate_tx` action indicates eight possible fields for data collection:

- `delegator_address` &mdash; The address from which ATOM will be redelegated (required).
- `source_validator_address` &mdash; The validator address from which ATOM will be redelegated (required).
- `destination_validator_address` &mdash; The validator address to which ATOM will be redelegated (required).
- `amount` &mdash; The amount of ATOM to be redelegated (required).
- `memo` &mdash; A message to record on-chain with your transaction (optional).
- `gas_limit` &mdash; Total amount of gas you're willing to let this transaction consume (optional).
  - If you do not include a gas limit, the Staking API uses a Cosmos library to select a good default for you.
- `gas_price` &mdash; how many ATOM you're willing to pay per unit of gas consumed by this transaction (optional).
  - If you do not include a gas price, the Staking API uses a Cosmos library to select a good default for you.
- `pubkey` &mdash; The public key value will be retrieved from the chain, however if you're using a fresh account (that does not have any outgoing activity) then this information is not available and it requires the user to pass it.
