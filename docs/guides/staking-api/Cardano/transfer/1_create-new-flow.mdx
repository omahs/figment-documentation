To initiate the transfer process, create a new flow with a `POST` request to `/api/v1/flows`.

### Parameters

- `flow` &mdash; `object`
  - `network_code` &mdash; `string` Network this flow operates on (ex. `cardano`) (required).
  - `chain_code` &mdash; `string` Chain this flow operates on (ex. `preprod`) (required).
  - `operation` &mdash; `string` The operation to perform (ex. `transfer`) (required).
  - `version` &mdash; `string` API version to use (ex. `v1`) (optional).

#### Sample cURL Request

```bash {5-10}
curl -X POST 'https://cardano-slate.datahub.figment.io/api/v1/flows' \
-H 'Content-Type: application/json' \
-H 'Authorization: <api_key>' \
-d '{
  "flow": {
    "network_code": "cardano",
    "chain_code": "preprod",
    "operation": "transfer",
    "version": "v1"
  }
}'
```

### Response

The response indicates one possible action:

- `create_transfer_tx`

The `create_transfer_tx` action indicates seven possible fields for data collection:

- `from_account` &mdash; The address from which ATOM will be transferred.
- `to_account` &mdash; The address to which ATOM will be transferred.
- `amount` &mdash; The amount of ATOM to be transferred.
