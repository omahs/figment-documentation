---
title: Create New Flow
---

To initiate the transfer process with Staking API firstly we need to create a new Flow with `POST /api/v1/flows`.

### Parameters

- `flow` - `object`
  - `network_code` - `string` Network this flow operates on (ex. near) (required)
  - `chain_code` - `string` Chain this flow operates on (ex. mainnet) (required)
  - `operation` - `string` The operation to perform (ex. staking) (required)
  - `version` - `string` API version to use (ex. v1) (optional)

#### Sample cURL Request

```json
curl -X POST 'https://near-slate.datahub.figment.io/api/v1/flows' \
--header 'Content-Type: application/json' \
--header 'Authorization: <api_key>' \
--data '{
  // highlight-start
  "flow": {
    "network_code": "near",
    "chain_code": "testnet",
    "operation": "transfer",
    "version": "v1"
  }
  // highlight-end
}'
```

### Response

- `id` - `string` ID of the newly created flow.
- `state` - `string` The current state of the flow.

The response indicates one possible action:

- `create_transfer_tx`

The `create_transfer_tx` action indicates seven fields for data collection:

- `from_account_address` (required) — The address which NEAR will be transferred from
- `from_account_pubkey` (required) — The account that will sign the transaction
- `to_account_address` (required) — The address which NEAR will be transferred to
- `amount` (required) — The amount of NEAR to be transferred
