The Figment API uses API keys to authenticate requests.

## Where to find the API Key

We're in the process of having all our services use the same API key. In the meantime:

- For requests to the Rewards API or Staking API, manage your API key on [https://datahub.figment.io](https://datahub.figment.io).
- For requests to the Validators API, get an API key from [https://hubble.figment.io/prime](https://hubble.figment.io/prime).

## How to pass the API Key
