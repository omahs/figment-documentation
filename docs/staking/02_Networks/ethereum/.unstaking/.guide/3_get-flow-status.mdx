To get the current state of the unstaking/existing flow, send a `GET` request to `/api/v1/flows/[:flow_id]` using the flow ID from the previous step.

### Parameters

- None

#### Sample cURL Request

```bash
curl -X GET 'https://eth-slate.datahub.figment.io/api/v1/flows/5d3fb8e0-bf2d-4fcc-9ff8-ed9d02d47659' \
-H 'Authorization: <api_key>'
```

### Response

- `state` &mdash; `string` The current state of the flow.

:::info note

The `state` field will update from `broadcasting` to `exiting`, then `withdrawing` and finally `unstaked`.

:::
